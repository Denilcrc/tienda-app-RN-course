# Products App

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```
2. Clonar .env.template a .env y agregar la direccion ip

3. Start the app

   ```bash
   npx expo start
   ```
